## How to run a clean A/B test

A clean test changes only one variable: the audience.

Everything else stays the same.

This ensures you’re evaluating Slopeside’s audience signals, not unrelated factors.

---

## Recommended test setup

For the most reliable results:
- Use the same campaign objective
- Use the same creative
- Use the same budget
- Use the same campaign or ad set structure
- Change only the audience

One version should use your existing audience.  
The other should use a Slopeside audience.

---

## Step-by-step testing approach

1. Identify a campaign or ad set that already performs reasonably well  
2. Duplicate the setup  
3. Swap the audience in the duplicate with a Slopeside audience  
4. Launch both versions simultaneously  
5. Let Meta optimize without intervention  

Avoid making changes while the test is running.

---

## Using Slopeside audiences in tests

You can test Slopeside audiences:
- Directly
- As seed audiences for expansion or lookalikes

Choose one approach per test to keep results clear.

---

## Common questions

**Can I test multiple Slopeside audiences at once?**  
For initial testing, we recommend testing one audience at a time.

**Can I change creative during the test?**  
No. Changing creative resets learning and invalidates comparisons.
