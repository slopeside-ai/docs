## Best practices for testing and launch

The most common mistake teams make is changing too many variables at once.

Slopeside works best when evaluated using a clean, controlled test.

---

## How to test Slopeside correctly

For a valid test:
- Use the same campaign structure
- Use the same creative
- Use the same budget
- Change only the audience

This isolates the impact of Slopeside’s audience signals.

---

## What to avoid early on

Avoid:
- Combining multiple Slopeside audiences in one test
- Changing creative mid-test
- Expecting instant results without a learning phase

These behaviors make results hard to interpret.

---

## What “normal” looks like

Early performance may fluctuate as Meta learns. This is expected.

Focus on:
- Directional improvements
- Consistency over time
- Clear comparisons to your existing audiences

---

## If results aren’t clear

If you’re unsure how to interpret test results:
- Check that the test setup was clean
- Review the Testing section later in the User Guide
- Avoid making changes too quickly
