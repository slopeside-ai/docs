These are the mistakes that most commonly cause a Slopeside test to produce bad or inconclusive data. Read this before you launch.

## Test setup

Use an existing ad set as your control — not a new one.

Your control needs conversion history for the comparison to be meaningful. Duplicate your current best-performing ad set and swap only the audience. If both ad sets are new, both are in Meta's learning phase simultaneously and results will be unreliable.

Keep everything identical except the audience.

Same creative, same copy, same landing page, same budget, same campaign objective. The only variable is the audience. Changing anything else makes it impossible to know what caused a performance difference.

Use ABO (Ad Set Budget Optimization), not CBO.

Set your budget at the ad set level so each ad set receives its allocated spend. With CBO, Meta can shift budget toward whichever ad set it already favors early on, which skews the comparison before the test is valid.

Add your Slopeside audience as an Advantage+ suggestion.

In your ad set settings, keep Advantage+ (ASC+) turned on and add your Slopeside audience under Suggest an audience. Keep targeting broad. Slopeside works as a starting signal, not a hard constraint.

## During the test

Do not touch the campaign for at least 14 days.

Editing, pausing, or restarting an ad set resets Meta's learning phase. Let both ad sets run for the full test window.

Expect higher CPM on the Slopeside ad set.

This is normal and expected. You are reaching a more targeted, higher-intent audience. Higher CPM does not mean the audience is underperforming. Watch CPA.

Do not judge results in the first few days.

Meta's learning phase produces erratic results while the algorithm is still exploring. Results become meaningful after both ad sets have logged roughly 50 conversions and at least 7 days of data. For a full, reliable read, run for 14 days.

## Reading results

CPA is the primary metric.

Compare cost per acquisition (or cost per lead, depending on your objective) between the two ad sets. This is what determines whether Slopeside is working.

Secondary metrics to watch:

- Conversion rate (are clicks converting at a higher rate?)

- ROAS (for ecommerce accounts)

Ignore:

- CPM differences between ad sets

- CTR in isolation (a higher CTR that doesn't convert is not a win)
